# 👟 SyntaxWear - Tênis e Sneakers Online

A **SyntaxWear** é uma interface de e-commerce moderna e elegante, desenvolvida para proporcionar a melhor experiência de compra de tênis e sneakers online. O projeto foca em um design limpo, responsividade e uma navegação intuitiva.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada com o uso de:
  - **Variáveis CSS**: Para facilitar a manutenção de cores e fontes.
  - **Flexbox & Grid**: Para criação de layouts complexos e responsivos.
  - **Metodologia de Componentes**: Organização dos estilos por blocos (header, footer, cards).

---

## ✨ Funcionalidades

- [x] **Cabeçalho Dinâmico**: Menu de navegação e acesso rápido ao perfil e carrinho.
- [x] **Seção Hero**: Destaque principal com chamadas para ação (CTA).
- [x] **Categorias**: Filtro visual para modelos Casuais, Esportivos, Modernos e Futuristas.
- [x] **Grid de Produtos**: Exposição organizada dos modelos em destaque.
- [x] **Rodapé Completo**: Newsletter, redes sociais e links úteis de navegação.

---

## 📁 Estrutura do Projeto

A organização dos arquivos segue uma estrutura profissional para facilitar a escalabilidade:

```text
ecommerce-syntaxwear/
├── css/
│   ├── components/       # Estilos específicos de cada parte do site
│   │   ├── footer.css
│   │   ├── header.css
│   │   ├── hero.css
│   │   ├── product-category.css
│   │   └── product-grid.css
│   ├── base.css          # Estilos globais (fontes, botões, etc.)
│   ├── reset.css         # Reset de estilos padrão do navegador
│   └── variables.css     # Definição de cores e tokens de design
├── images/               # Ativos visuais (logos, ícones e fotos)
│   ├── banners/
│   ├── icons/
│   ├── logo/
│   └── products/
└── index.html            # Arquivo principal do site
```

---

## 🛠️ Como Executar o Projeto

Como este é um projeto front-end estático, você não precisa de nenhum servidor complexo.

1. Faça o download ou clone este repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox).
